# Layout widgets

Use this group to shape space before you worry about product details.

## Core structure

The most common primitives are:

- `Container`
- `Row` / `Column`
- `Scroll`
- `Spacer`
- `Grid` / `GridItem`
- `ZStack` / `Positioned`

## High-level layout helpers

When the pattern is already established in the repo, reach for:

- `HStack` / `VStack`
- `Center`
- `SimpleGrid`
- `SplitView`
- `Wrap`
- `AspectRatio`
- `SafeArea`

## Escape hatches

- `LayoutBuilder` when layout depends on current constraints
- `Transform` and `Clip` for composition boundaries
- `absolute_fill(...)` when a child should take the available stack bounds

Use the widget-gallery and chart/inbox examples to see which of these are already exercised in real screens.
