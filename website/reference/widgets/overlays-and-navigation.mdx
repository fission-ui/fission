# Overlays and navigation

These widgets all depend on the runtime's overlay, focus, or routing machinery.

## Overlay-oriented widgets

- `Modal`
- `Popover`
- `Drawer`
- `Tooltip`
- `Toast`
- `Menu`, `MenuButton`, `MenuItem`
- `Portal`
- `flyout(...)`

## Navigation-oriented widgets

- `Tabs` / `TabItem`
- `Accordion` / `AccordionItem`
- `Breadcrumb` / `BreadcrumbItem`
- `Pagination`
- `Route`, `RouteParams`, `Router`
- `Hero`

## Focus helpers

- `FocusScope`
- `Transition`

Use the inbox and editor examples when you need proof that a layered app shell already works in the repo.
