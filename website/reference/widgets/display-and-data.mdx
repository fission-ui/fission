# Display and data widgets

These widgets are most useful once layout and state flow are already settled.

## Display

- `Text`, `RichText`, `Code`, `Kbd`
- `Icon`, `Avatar`, `Badge`, `Tag`, `Link`
- `Card`, `Divider`, `EmptyState`

## Feedback

- `Alert`
- `ProgressBar`
- `CircularProgress`
- `Spinner`
- `Skeleton`
- `Stat`
- `Stepper`

## Data and structure

- `DataTable`, `TableColumn`, `TableRow`
- `Timeline`, `TimelineItem`
- `TreeView`, `TreeItem`
- `Calendar`

The widget gallery is the quickest breadth demo, while inbox and editor show how these widgets fit inside bigger products.
