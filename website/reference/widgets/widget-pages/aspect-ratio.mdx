# AspectRatio

Use `AspectRatio` when a child should keep a fixed width/height relationship inside the normal box layout system.

## Minimal usage

```rust
use fission_widgets::AspectRatio;

let node = AspectRatio {
    ratio: 16.0 / 9.0,
    child: Box::new(Text::new("Preview").into_node()),
}
.build(ctx, view);
```

## API surface

- `ratio`: `f32`
- `child`: `Box<Node>`

## Notes

The widget lowers to a box layout node with `aspect_ratio: Some(ratio)`, so the parent constraints still decide the final size.
