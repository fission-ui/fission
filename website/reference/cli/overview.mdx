# CLI overview

Standard commands:

`fission init <app>`

Creates and configures a new project scaffold.

`fission add-target <target>`

Adds a build/run target.

Use these commands consistently in local automation so every team member works from the same workflow.

Run targets from generated output scripts:

- Desktop: `cargo run`
- Web: `./platforms/web/run-browser.sh`
- iOS: `./platforms/ios/run-sim.sh` (after `cargo fission add-target ios`)
- Android: `./platforms/android/run-emulator.sh` (after `cargo fission add-target android`)
