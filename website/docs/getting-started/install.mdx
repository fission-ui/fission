# Install and setup

You can get started from your existing Rust environment.

First add the core crate to your project.

```bash
cargo add fission_core
```

Install the CLI so you can scaffold and run targets consistently.

```bash
cargo install fission-cli
# if you work from source:
# cargo install --locked --path crates/cli
```

Create a new app, then add targets. Add desktop and web by default, then include native mobile targets as needed.

```bash
fission init my-fission-app
cd my-fission-app
fission add-target desktop
fission add-target web
fission add-target ios
fission add-target android
```

Use `cargo run` for desktop, then validate web in the browser. Run `./platforms/ios/run-sim.sh` for iOS simulator checks and `./platforms/android/run-emulator.sh` for Android emulator checks.

```bash
cargo run
./platforms/web/run-browser.sh
./platforms/ios/run-sim.sh
./platforms/android/run-emulator.sh
```

If these run, continue to [Build your first app](/docs/getting-started/first-app) and keep this sequence fixed for your first week.

If `fission` fails to resolve, confirm your Rust toolchain and PATH before continuing.
