# Chart catalog

This catalog is the shape of Fission Charts as a product surface. Entries marked **Available now** are implemented in the current native chart renderer. Entries marked **Next implementation slice** are the nearest expansion work. Entries marked **Planned catalog target** are documented here so the framework grows toward a coherent, complete charting system instead of accumulating one-off chart demos.

Each card includes a chart-only screenshot, the data shape the chart expects, and the product situation where the chart belongs. That is deliberate. A chart reference should help you choose the right visualization before you start writing code.
