# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in Fission, please report it responsibly.

**Do not open a public issue.** Instead, email security concerns to the maintainers directly via the contact information in the repository.

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days for critical issues.
